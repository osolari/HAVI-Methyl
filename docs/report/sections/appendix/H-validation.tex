\section{Simulator validation}
\label{app:simvalidation}

This appendix records the intended simulator-validation axes and current artifact status. The validation numbers should be regenerated from code before final publication and should be accompanied by plots.

\subsection*{Fragment-length distribution}
The simulator should reproduce a dominant mononucleosomal mode near $167$~bp and secondary modes near multi-nucleosomal lengths. Current sidecar files contain planning/check values, but the publication version should regenerate the histogram, reference overlay, and distance metric from the exact simulator used for experiments.

\subsection*{End-motif frequencies}
The simulator should match broad plasma-WGS end-motif distributions, including enrichment of common 4-mers at cut sites. Current values should be treated as validation targets or preliminary checks until regenerated against the final motif model.

\subsection*{Periodicity spectrum}
Cut-site autocorrelation should exhibit periodicity near the helical pitch of DNA. The final validation bundle should report the peak location, uncertainty or resolution, and sensitivity to the nucleosome-positioning parameters.

\subsection*{Methylation-conditioned cut bias}
The simulator should show methylation-conditioned shifts in cut-site or motif preferences because this is the signal exploited by FinaleMe-style and HAVI-Methyl predictors. The final validation should include a stratified plot and a numeric effect-size table generated by the released code.
